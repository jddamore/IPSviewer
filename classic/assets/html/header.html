International Patient Summary (IPS) Viewer for Connectathon

Links to published Implementation Guide and the latest CI build
Please note that this tool is an open-source project under development. It only renders the following sections of IPS bundles: Advance Directives, Allergies, Immunizations, Medications, Problems and Results. Rendering of FHIR resources may be incomplete and using the narrative may be necessary in some circumstances.
